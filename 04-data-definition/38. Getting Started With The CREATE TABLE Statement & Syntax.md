# 38. Getting Started With The CREATE TABLE Statement & Syntax

We need to create the following tables:

- employers
  - Company
  - Name
  - Address (Street, City)
  - Yearly Revenue
  - Is Hiring?
- users
  - Name (Full Name)
  - Yearly Salary
  - Status (Employed?)
  - Employer
- conversations
  - User
  - Employer
  - Message
  - Date Sent
